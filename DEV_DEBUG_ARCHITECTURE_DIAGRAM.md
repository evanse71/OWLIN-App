## Component Responsibilities

### Frontend (DevDebug.tsx)

| Responsibility | Implementation |
|----------------|----------------|
| Render UI | React component with Tailwind CSS |
| Manage state | useState for issues, selection, explanation |
| API calls | fetch() to backend endpoints |
| User interactions | onClick handlers, input changes |
| Display results | Color-coded cards, copy buttons |

### Backend (dev_tools.py)

| Responsibility | Implementation |
|----------------|----------------|
| Route requests | FastAPI router with @router.get/post |
| Validate input | Pydantic models (ExplainRequest) |
| Coordinate | Calls CheckRunner and LLMExplainer |
| Return responses | Pydantic models (RunChecksResponse, ExplainResponse) |

### CheckRunner (runner.py)

| Responsibility | Implementation |
|----------------|----------------|
| Execute tools | subprocess.run() for each tool |
| Parse output | Regex and JSON parsing |
| Structure issues | Creates CodeIssue objects |
| Handle errors | Try/catch with error list |

### LLMExplainer (llm_explainer.py)

| Responsibility | Implementation |
|----------------|----------------|
| Check Ollama | HTTP request to Ollama API |
| Generate AI explanation | POST to Ollama with prompt |
| Fallback templates | Tool-specific template methods |
| Format output | Creates IssueExplanation object |
